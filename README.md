# 🎬 Movie Theater Booking System

## 📌 Project Description

The Movie Theater Booking System is a web-based application that allows users to browse available movies and book tickets online.
The system provides an interactive interface for viewing movie details, selecting showtimes, and managing bookings.

This project demonstrates the use of web technologies to build a simple movie ticket booking platform.

## 🚀 Features

* User login system
* Movie browsing dashboard
* Movie information pages
* Show date selection
* Ticket booking interface
* Image gallery
* Contact page
* Facilities
* Services

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
